Métodos Then e Catch


A função then e catch são métodos disponiveis em objetos Promise em JavaScript, que são usados para lidar como o resultado (resolução) ou erro (rejeição) de uma Promise.

  1. then ( ) :
    • O método then ( ) é usado para registrar uma ou mais funções de retorno de chamada que serão executadas com sucesso.
    • Ele recebe um ou dois argumentos: a função de retorno de chamada para o caso de sucesso e opcionalmente uma função de retorno de chamada em caso de erro.
    • Se a Promise for resolvida com sucesso, a função de retorno de chamada passada para then ( ) será invocada com valor resultante como argumento.
    • Se a Promise for rejeitada, o método then ( ) não será executado, e a execução será passada para o próximo método catch

  2. Catch ( )
    • O método Catch é usado para registar uma função de retorno de chamada que será executada somente se a Promise for rejeitada.
    • Ele recebe uma função de retorno de chamada como argumento, que será invadada com o motivo do erro (geralmente um Bojeto Error) como argumento.
    • O método catch é usado para lidar com erros que ocorrem dutante a execução de uma Promise.

    3. Em resumo, o método then() é usado para lidar com o caso de sucesso da Promise, enquanto o método catch() é usado para lidar com o caso de erro da Promise. Ambos os métodos permitem que você escreva código mais limpo e legível ao lidar com operações assíncronas em JavaScript.
