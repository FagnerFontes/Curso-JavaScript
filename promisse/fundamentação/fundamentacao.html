Promise


Uma Promise no JavaScript é um objeto que representa a eventual conclusão (ou falha) de uma operação assíncrona. É comumente usada para trabalhar com código assíncrono de forma limpa e eficiente, evitando o chamado callback hell aninhamento excessivo de callbacks e facilitando o tratamento de erros.

A Promisse pode estar em três estados diferentes:

  1. Pending : (Pendente): Estado inicial, quando a Promisse está em execução, mas ainda não foi concluída nem rejeitada.
  2. Fulfiled : (Concluída) : Quando a operação assíncrona é bem-sucedida, ou seja, a Promise é resolvida com um valor.
  3. Rejected : (Rejeitada) : Quando uma operação assíncrona falha, a Promise é rejeitada e é gerado um erro.

Depois de criar uma promisse, podemos encadear chamadas de método then ( ) e catch( ) para lidar com a resolução ou rejeição da Promise.
